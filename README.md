# Pictorial Symbols Icon Library

A browsable, searchable SVG library of pictogram icons digitized from a printed reference book.

**Live viewer:** open `index.html` (or the GitHub Pages URL for this repo) — search by name, filter by category,
click any icon for a larger preview, download, or star it for the session.

## Source & attribution

These icons are hand-digitized (scan → crop → vector trace → name) from:

> *Handbook of Pictorial Symbols: 3,250 Examples from International Sources*, Rudolf Modley with the assistance
> of William R. Myers. Dover Publications, Inc., New York, 1976. ISBN 0-486-23357-X.

The book's own copyright page states that *"up to ten illustrations from this book... may be reproduced on any
one project or in any single publication, free and without special permission,"* provided a credit line is
included. This repository contains far more than ten derived icons and is **not** covered by that license — it
exceeds the publisher's stated permitted use. Treat this repo as a personal/archival digitization project rather
than a freely licensed icon set, and don't redistribute it as if it were open-licensed. If you want to use more
than a handful of these icons in your own published work, contact Dover Publications for permission.

## What's here

- `*.svg` — one file per distinct icon design (vtracer polygon trace of the printed silhouette)
- `index.csv` — filename, display name, and source page/row/column for every icon
- `index.html` — the search/filter/lightbox browser
